# 02. BFS (너비 우선 탐색)

## 개념
시작점에서 **가까운 곳부터** 한 겹씩 퍼져나가며 탐색. **큐(queue)** 사용. 방문 시 거리를 기록하면 **가중치가 모두 1인 그래프의 최단거리**를 구할 수 있다.

## 언제 쓰나
- "최소 이동 횟수", "최단 거리" (간선 비용이 모두 같을 때)
- 격자(미로)에서 한 칸씩 이동하는 최단경로
- 동시에 여러 곳에서 퍼지는 문제 (다중 시작점 BFS — 토마토, 불 번지기)

## C++ 템플릿 (격자 BFS)
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, m;
int board[1001][1001];
int dist[1001][1001];
int dx[4] = {0,0,1,-1};
int dy[4] = {1,-1,0,0};

void bfs(int sx, int sy) {
    queue<pair<int,int>> q;
    q.push({sx, sy});
    dist[sx][sy] = 1;                 // 시작점 거리(문제 정의에 맞게 0 또는 1)
    while (!q.empty()) {
        auto [x, y] = q.front(); q.pop();
        for (int d = 0; d < 4; d++) {
            int nx = x + dx[d], ny = y + dy[d];
            if (nx < 0 || ny < 0 || nx >= n || ny >= m) continue; // 범위 밖
            if (dist[nx][ny] != 0) continue;        // 이미 방문
            if (board[nx][ny] == 0) continue;       // 벽 등 이동 불가
            dist[nx][ny] = dist[x][y] + 1;          // 거리 갱신 = 방문 처리
            q.push({nx, ny});
        }
    }
}
```

## 시간복잡도
- O(V + E) — 정점 + 간선. 격자면 O(N·M).

## 함정
- **방문 체크는 큐에 넣을 때** 한다 (꺼낼 때 하면 중복 삽입으로 TLE/메모리 폭발).
- 거리 배열을 방문 배열로 겸하면 코드가 깔끔 (`dist != 0` = 방문).
- 가중치가 서로 다르면 BFS 아님 → **다익스트라**(06/07 이후 Gold).
- `auto [x,y] = q.front();` 구조적 바인딩은 C++17 (백준 `-std=gnu++17` OK).
