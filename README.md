# 알고리즘 스터디 (백준 / C++)

> 재가동 모드: Gold 60문제 경험자의 감 되찾기 → Bronze 워밍업 → Silver → Gold

## 진행 방식 (레벨별 2단계)

**"문제 생성해줘"** 라고 말하면 → Claude가 현재 레벨/주제에 맞는 **실제 백준 문제**를 제시.
문제 선정 기준: **Formal**(정석 형식) + **대표 문제**(알고리즘의 canonical) + **알아야 하는 문제**(필수).

### 🩹 Bronze ~ Silver 2 — 재활 모드
- `problems/`에 **길라잡이 C++ 파일** 생성: 자료구조/입출력 골격 + `// TODO` 빈칸 + 단계별 주석 힌트
- 빈칸을 채우며 감 되찾기 → 막히면 "힌트"로 추가 설명

### ⚔️ Silver 1 ~ Gold — 실전 모드
- **문제 설명 + 예시 입출력** 제공 (스캐폴드 없이 백지에서)
- **레퍼런스 풀이 블로그 주소 필수** (Claude가 웹 검색으로 실제 존재하는 URL 검증 후 제공 — 막혔을 때 참고용)

### 공통
1. 직접 풀고 → **백준에 제출**
2. 결과 보고 ("맞았어" / "틀렸어" / "막혔어")
   - 맞으면 → 코드 리뷰 + progress.md 기록 + 다음 문제
   - 막히면 → 힌트 단계별 (정답 코드는 "정답 보여줘" 요청 시에만)

## 명령어

| 입력 | 동작 |
|------|------|
| `문제 생성해줘` | 다음 문제 1개 제시 |
| `문제 생성해줘 (주제)` | 특정 주제 문제 (예: "문제 생성해줘 BFS") |
| `힌트` | 현재 문제 힌트 단계별 제공 |
| `정답 보여줘` | 모범 풀이 + 해설 |
| `리뷰` | 내 코드 리뷰 |
| `진도` | progress.md 현황 확인 |

## 로드맵

### Phase 0 — Bronze 워밍업 (C++ I/O & 기본기 재장착)
- 빠른 입출력(`ios_base::sync_with_stdio(false)`), 배열, 반복문, std::sort
- 목표: 2~3문제로 감 되찾고 즉시 Silver 진입

### Phase 1 — Silver 핵심
- **자료구조**: 스택/큐/덱 (10828, 10845, 10866), 우선순위큐
- **정렬/탐색**: 좌표 정렬(11650), 이분탐색(1920, 10816)
- **BFS/DFS**: 연결요소(2667, 2178), 그래프 탐색
- **그리디**: 1931 회의실배정, 11399 ATM
- **DP 입문**: 1463, 9095, 11726, 11053(LIS)

### Phase 2 — Gold 진입
- **BFS 응용**: 7576 토마토, 2206 벽부수기
- **DP 중급**: 12865 배낭, 11054, 2565
- **그래프**: 다익스트라(1753), 위상정렬(2252), MST(1197)
- **이분탐색 응용**: 2110 공유기 설치, 매개변수 탐색

  silver/
  gold/
study_materials/         # 알고리즘 개념 정리 (정렬/BFS/DFS/구현/그리디/이분탐색/DP)

파일명 규칙: `문제번호_문제이름.cpp`
> 전체 풀이 순서는 [문제집.md](문제집.md) 참고.

## 로컬 빌드 환경 (Windows / g++)
- 컴파일러: **MinGW-w64 g++ 16.1.0** (WinLibs, PATH 등록 완료)
- VS Code에서 `.cpp` 열고 **`Ctrl+Shift+B`** → 컴파일 후 같은 폴더 `input.txt`로 자동 실행
  - 예제를 `input.txt`에 넣어두면 출력이 터미널에 바로 나옴 → 백준 제출 전 검증
- 수동 컴파일: `g++ -O2 -std=gnu++17 -static -o a.exe 파일.cpp`
  - ⚠️ `-static` 권장 (WinLibs 동적 DLL 의존성 회피, 단일 exe로 실행)
